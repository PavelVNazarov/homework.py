Объяснение кода:
Импорт и создание приложения:

Python
Копировать
from fastapi import FastAPI

app = FastAPI()
Импортируется класс FastAPI, который используется для создания приложения и управления маршрутизацией.
Маршрут к главной странице ("/"):

Python
Копировать
@app.get("/")
async def main_page() -> dict:
    return {"message": "Главная страница"}
Определяет маршрут для главной страницы, который возвращает сообщение "Главная страница".
Маршрут к странице администратора ("/user/admin"):

Python
Копировать
@app.get("/user/admin")
async def admin_page() -> dict:
    return {"message": "Вы вошли как администратор"}
Определяет маршрут для страницы администратора, возвращая сообщение об успешном входе.
Маршрут к страницам пользователей ("/user/{user_id}"):

Python
Копировать
@app.get("/user/{user_id}")
async def user_page(user_id: int) -> dict:
    return {"message": f"Вы вошли как пользователь № {user_id}"}
Использует переменную маршрута user_id, которая передается в функцию и возвращается в сообщении.
Маршрут для передачи данных о пользователе ("/user"):

Python
Копировать
@app.get("/user")
async def user_info(username: str, age: int) -> dict:
    return {"message": f"Информация о пользователе. Имя: {username}, Возраст: {age}"}
Позволяет передать username и age в качестве параметров запроса и возвращает информацию об этом пользователе.
Запуск приложения
Для запуска приложения сохраните код в файл, например, main.py, и выполните команду в терминале:

Bash
Копировать
python -m uvicorn main:app --reload
Затем откройте в браузере http://127.0.0.1:8000/docs, чтобы воспользоваться интерфейсом Swagger, который автоматически сгенерирован FastAPI и позволяет тестировать Ваши маршруты.
