Объяснение кода:
Импорт библиотек:

FastAPI — основной класс для создания приложения.
Path — используется для валидации параметров пути.
Annotated — позволяет добавлять метаданные к параметрам.
Создание приложения:

app = FastAPI() — создается экземпляр приложения FastAPI.
Маршрут главной страницы:

@app.get("/") — определяет маршрут для главной страницы.
async def main_page() — асинхронная функция, возвращающая сообщение "Главная страница".
Маршрут для администратора:

@app.get("/user/admin") — определяет маршрут для страницы администратора.
async def admin_page() — возвращает сообщение "Вы вошли как администратор".
Маршрут для пользователей с валидацией:

@app.get("/user/{user_id}") — маршрут, принимающий user_id.
user_id: Annotated[int, Path(ge=1, le=100, description="Enter User ID", example=1)] — параметр user_id должен быть целым числом от 1 до 100 с описанием и примером.
Маршрут для информации о пользователе с валидацией:

@app.get("/user/{username}/{age}") — маршрут, принимающий username и age.
username: Annotated[str, Path(min_length=5, max_length=20, description="Enter username", example="UrbanUser")] — параметр username должен быть строкой длиной от 5 до 20 символов.
age: Annotated[int, Path(ge=18, le=120, description="Enter age", example=24)] — параметр age должен быть целым числом от 18 до 120.
